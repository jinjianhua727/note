Math对象简介

Math对象虽然是js提供的原生对象，但是不像Date对象等，它不是构造器，Math对象的属性和方法都是静态。

Math对象提供数学公式和数学常量，与用js直接编写的计算功能相比，Math对象提供的计算功能要快很多。

常用的Math对象属性与方法有PI、三角函数、求最值、舍入方法、随机数，除了这些Math对象还提供了自然对数、平方根等相关的属性和方法。

PI

Math.PI：圆周率，圆的直径和周长之比。

PI经常用于求取弧度，一个圆（角度为360°）的弧度为2 * Math.PI，根据这个值可以求出角度对应的弧度，弧度= 角度 * Math.PI / 180;如直角90°对应的弧度为Math.PI/2。

三角函数

sin(), cos(), tan(),asin(), acos(), atan(), atan2()。

参数为弧度。参考数学三角函数公式获取想要的长度。

如直角三角形，角度为30，斜边长度为4，求取对边长度

var c = 4;
var rad = 30 * Math.PI / 180;
var a = Math.sin(rad) * c;
// 1.9999999999999998，由数学公式计算结果是2
console.log(a)

求最值

Math.min()，Math.max()。

参数是需要比较大小的数值。通过应用apply可以求取数组元素的最值。

var num1 = 1;
var num2 = 2;
var num3 = 3;
var arr = [1,2,3];
// 3
console.log(Math.max(num1,num2,num3))
// 1
console.log(Math.min.apply({},arr))

舍入方法

Math.ceil()：向上舍入

Math.floor()：向下舍入

Math.round()：四舍五入

// 3
console.log(Math.ceil(2.8))
// 2
console.log(Math.floor(2.8))
// 3
console.log(Math.round(2.8))

根据需求选择合适的舍入方法，如求取一个数组百位数的数字时，要向下舍入。

随机数

Math.random()。返回一个大于等于0小于1的随机数，下面方法返回某个整数范围的随机值。

// 1或2或3
console.log(rd(1,3))
function rd(min,max){
	// 左闭右开
	// return Math.floor(Math.random() * (max - min) + min);
	// 左开右闭
	// return Math.floor(Math.random() * (max - min) + min + 1);
	// 左右都开
	// return Math.floor(Math.random()*(max - min - 1)) + min +1;
	// 左右都闭合
	return Math.floor(Math.random() * (max - min + 1) + min);
}
