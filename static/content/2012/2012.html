一个域名的组成包括

协议：http://

子域名：www

主域名：baidu.com

端口号：8080

请求资源地址：js/jquery.js

当协议、子域名、主域名、端口号任一不同时，都算不同域，不同域之间请求资源就算跨域。

js出于安全方面的考虑，不允许跨域调用其它页面的对象。

解决跨域的方案

1.代理，通过在同域名的web服务器端传建一个代理

2.JSONP，可以解决主流浏览器跨域GET请求。

JSONP实现思路是利用script的特性，

在a域下

<script>
	function jsonp(json){
		alert(json["name"])
	}
	alert()
</script>
<script src="http://www.bbb.con/jsonp.js"></script>

在b域下

jsonp({"name":"xiaojin"})

<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>a域取得b域的json</title>
	<script src="http://cdn.staticfile.org/jquery/1.11.2/jquery.min.js"></script>
</head>
<body>
	<h1>********* 同域ajax **********</h1>
	<button id="btn">跨域获取</button>
	<h1 id="h1">获取的内容是：</h1>
	<script>
		$("#btn").click(function(){
			$.ajax({
				type:"get",
				url:"http://localhost:5432/github/note/content/2012/demo1.json",
				dataType:"json",
				success:function(data){
					$("#h1").text(data.name);
				},
				error:function(){
					$("#h1").text("发生错误")
				}
			})
		})
	</script>
	<h1>******** 跨域jsonp ********</h1>
	<button id="btn2">跨域获取</button>
	<h1 id="h2">获取的内容是：</h1>
	<script>
		$("#btn2").click(function(){
			$.ajax({
				type:"get",
				url:"http://192.168.1.103:5432/github/note/content/2012/demo1.php",
				dataType:"jsonp",
				jsonp:"callback",
				success:function(data){
					$("#h2").text(data.name);
				},

					$("#h2").text("发生错误")
				}
			})
		})
	</script>
</body>
</html>
